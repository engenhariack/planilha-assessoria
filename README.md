# planilha-assessoria
Planilha para assessoria de corrida + cartões de treino com IA generativa

Criei uma planilha para prescrição de treinos de corrida em Excel, escalável e com valor agregado, para um assessor de corrida. Uma alternativa às plataformas pagas!

Para melhor experiência do aluno em relação a visualização e entendimento dos treinos, criamos cartões de treino. O prompt de criação permite criar cards de forma escalável no mesmo padrão com base nos treinos da planilha, de forma gratuita! São criados com DALLE-3, a IA generativa de imagens dentro do Chat-GPT, da Open AI.

**`Você pode visualizar os arquivo "Desenvolvimento de Planilha + Desenvolvimento de Planilha + Cartões com IA generativa" e "Cartões de corrida" dentro desse repositório.`**
